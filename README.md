# Wi-Fi Deauthentication Attack Detector
A low-cost, hardware-based wireless intrusion detection system (WIDS) built using an ESP8266 microcontroller.

## 📌 Project Overview
This project configures an ESP8266 chip to run in promiscuous mode, allowing it to passively sniff ambient 802.11 management frames. If a Wi-Fi deauthentication or disassociation attack is detected in the area, the device triggers a physical alert (LED/OLED) and logs the attacker's MAC address.

### Technical Skills Demonstrated:
* **Embedded Systems:** Microcontroller programming via Arduino IDE (C++)
* **Wireless Networking:** 802.11 frame analysis, promiscuous mode sniffing
* **Cybersecurity:** Network defense, Wi-Fi vulnerability analysis, threat detection

## 🛠️ Hardware Requirements
* NodeMCU ESP8266 Board (~$3–$5)
* Micro-USB to USB Cable (Data transfer capable)
* Optional: 0.96" I2C OLED Display (SSD1306)
